# Claude Code Guidelines

## Language

Always use British English spelling in all code, comments, documentation,
and user-facing strings. For example: "recognised" not "recognized",
"behaviour" not "behavior", "summarising" not "summarizing",
"prioritised" not "prioritized", "specialised" not "specialized",
"initialised" not "initialized", "analyse" not "analyze",
"sanitise" not "sanitize", "colour" not "color".
